About the Project

The Smart Waste Management System is designed to improve waste collection efficiency using modern web technologies. It helps monitor waste levels, optimize garbage collection routes and promote eco-friendly waste disposal.

Key Features

Smart Bin Monitoring

Detect waste level in bins to prevent overflow.

Route Optimization

Improve waste collection routes for efficiency.

Eco Friendly

Encourage recycling and proper waste segregation.

Real Time Updates

Monitor waste data through a web dashboard.
